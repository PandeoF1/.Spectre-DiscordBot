</p>

A NodeJS Discord Bot for the .Spectre team. 

## **How to install the bot**

> To install the bot, run the next command:

    $ npm install

## **How to run the bot**

> To run the bot, simply open the terminal of your choice and run the following from repository root:

    $ npm start run
